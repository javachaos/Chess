Rust + WebAssembly

Chess Engine

The desktop app now has a browser twin. GitHub Pages serves the same Rust engine compiled to WebAssembly, wrapped in a lighter web shell.

Preparing board

Loading engine...

The WebAssembly bundle only needs a moment. Once it finishes loading, the full board and control panel appear inside this frame.

Browser builds search on one thread, so higher depths can feel heavier than the native desktop version.

How to play

Same board, lighter shell

Choose your color in the right-side controls, click a piece, then click a highlighted square. The in-browser UI is the same shared egui app used by the desktop build.

Deployment

Made for GitHub Pages

A GitHub Actions workflow builds the wasm library, runs `wasm-bindgen`, and publishes a static site artifact. No custom server is required.

Performance note

Great for play, not for benchmarks

Opening moves stay snappy thanks to the book, but deep off-book searches still run faster in the packaged desktop app where native threads are available.
